I'm a full-stack engineer who owns features from the database all the way to production. I work day to day across **Next.js**, **Nest.js**, **React** and **React Native**, and I don't stop at writing the code — I deploy and operate it myself on **AWS** behind **Nginx** on **Ubuntu** servers.

- 🔭 Currently building and maintaining production web apps at **[Caffstack](https://caffstack.com)**
- ⚙️ I deploy and run Next.js / Nest.js apps in production on **AWS EC2**, fronted by an **Nginx** reverse proxy on **Ubuntu Linux**
- 🧱 Comfortable across the whole stack: typed APIs, relational & document databases, modern React, and CI-friendly deployments
- 📱 Cross-platform mobile with **React Native**
